### üêõ Troubleshooting
*   **Import Errors:** If you try to run locally and get errors about `WhisperProcessor` or `torch`, your local environment has version conflicts. This is why we recommend the Colab method above.
*   **Submission Time Limit:** The competition allows 2 hours for inference. The `whisper-medium` model should fit comfortably within this limit.
